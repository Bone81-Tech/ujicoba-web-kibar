Program Kerja KIBAR

Agenda kerja jangka pendek dan jangka panjang yang selaras dengan kebijakan pemerintah daerah.

Komunitas Independen Bersama Azaz Rakyat (KIBAR) telah menyusun agenda kerja yang diterapkan oleh setiap fungsi dalam jajaran organisasi. Berikut adalah rincian program kerja kami:

Dewan Kehormatan

Fungsi: Memberikan nasihat dan pertimbangan kepada Dewan Komite Nasional.

Program Jangka Pendek:

  • Menyusun program kerja
  • Membuat sistem kerja

Program Jangka Panjang:

Diharapkan diketuai oleh Presiden RI.

Dewan Pembina

Fungsi: Memberikan pengarahan/pembinaan terhadap lembaga.

Program Jangka Pendek:

  • Menyusun program kerja
  • Membuat sistem kerja

Program Jangka Panjang:

Diharapkan diketuai oleh pejabat setingkat Menteri.

Dewan Pakar

Fungsi: Memberikan masukan dan pertimbangan kepada DKPN.

Program Jangka Pendek:

  • Menyusun program kerja
  • Membuat sistem kerja

Program Jangka Panjang:

Menjadi tenaga tetap di LP2KP dengan fasilitasnya.

Dewan Pengawas

Fungsi: Melaksanakan pengawasan sesuai agenda kegiatan nasional.

Program Jangka Pendek:

  • Membuat sistem pengendalian internal
  • Melaksanakan fungsi internal control
  • Menangani kasus client

Program Jangka Panjang:

  • Membangun LP2KP sebagai lembaga prestisius
  • Mengupayakan Kepres/SK Mendagri untuk LP2KP sebagai lembaga mitra pemerintah
  • Menjalin hubungan dengan NGO internasional dan negara pemberi bantuan ke Indonesia

Divisi Advokasi

Fungsi: Melaksanakan pengawasan internal, perlindungan hukum, dan advokasi.

Program Jangka Pendek:

  • Menyusun program kerja
  • Membuat sistem kerja

Program Jangka Panjang:

  • Mendirikan Komunitas Konsultan/Biro Bantuan Hukum terpercaya
  • Memperkuat payung hukum untuk komunitas dan pengurus KIBAR

Divisi SDM

Fungsi: Memeriksa kinerja dari DKPN, KPP, dan KPD.

Program Jangka Pendek:

  • Membuat sistem pengendalian internal bersama Legal Executive
  • Melaksanakan pengawasan, penelitian, dan penyidikan pelanggaran pengurus

Program Jangka Panjang:

  • Mengimplementasikan program kerja jangka panjang DKPN sesuai bidangnya
  • Masuk secara struktural dalam lembaga sebagai tenaga tetap

Divisi Litbang

Fungsi: Melaksanakan program kerja sesuai bidangnya.

Program Jangka Pendek:

  • Menyusun program kerja jangka pendek dan panjang
  • Membuat sistem dan mekanisme kerja

Program Jangka Panjang:

Melaksanakan program jangka panjang DKI sesuai bidangnya.

Divisi- Divisi

Fungsi: Melaksanakan program kerja sesuai bidangnya.

Program Jangka Pendek:

  • Menyusun program kerja jangka pendek dan panjang
  • Membuat sistem dan mekanisme kerja

Program Jangka Panjang:

Melaksanakan program jangka panjang DKI sesuai divisinya.

Divisi Perencanaan Program

Fungsi: Melaksanakan fungsi audit kinerja pemerintahan provinsi.

Program Jangka Pendek:

  • Menyusun program kerja jangka pendek dan panjang
  • Membuat sistem dan mekanisme kerja
  • Melengkapi susunan kepengurusan
  • Menjalin hubungan kemitraan dengan gubernur
  • Melakukan audit kinerja pemerintahan provinsi
  • Membuat laporan hasil audit kepada gubernur

Program Jangka Panjang:

  • Menjadi komunitas prestisius di wilayah kerjanya
  • Diberikan kepercayaan oleh pemerintah daerah tingkat I untuk merekrut PNS dan melakukan fit & proper test pejabat eselon II dan III
  • Mendirikan media massa dengan nama KIBAR News atau setara

Divisi Humas dan Komunitas

Fungsi: Melaksanakan fungsi audit kinerja pemerintahan kabupaten/kota.

Program Jangka Pendek:

  • Menyusun program kerja jangka pendek dan panjang
  • Membuat sistem dan mekanisme kerja
  • Melengkapi susunan kepengurusan
  • Menjalin hubungan kemitraan dengan bupati
  • Melakukan audit kinerja pemerintahan kabupaten/kota
  • Membuat laporan hasil audit kepada bupati

Program Jangka Panjang:

  • Menjadi komunitas prestisius di wilayah kerjanya
  • Diberikan kepercayaan oleh pemerintah daerah tingkat II untuk merekrut PNS dan melakukan fit & proper test pejabat eselon II dan III
